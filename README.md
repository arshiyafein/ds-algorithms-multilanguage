# Data Structures & Algorithms Engineering Project

This project demonstrates fundamental **data structures** and **algorithms** implemented in **Python, C++, and Java**.

## Structure

- `python/`: Python implementation
  - `structures/`: Stack, Queue, Linked List, BST
  - `algorithms/`: Sorting, Searching, Graph Algorithms
  - `analysis/`: Performance tests
- `cpp/`: C++ implementation of BST and Graph
- `java/`: Java implementation of Stack, Queue, and Linked List
- `benchmarks/`: Performance comparison results

## How to Run

### Python
```bash
python3 python/structures/stack.py
python3 python/algorithms/sorting.py
